🤔 Why does this happen?

R1's link to A failed. R2 advertises "A is 2 away" to R1. R1 thinks "Great! I can reach A through R2 with cost 3!" But R2's path goes through R1. R2 doesn't share the full path — only (destination, cost). So R1 doesn't realize R2's path is useless.

✅ Poison Reverse Saves the Day!

With poison reverse, R2 tells R1: "A is infinity from me" (because R2 learned about A via R1). When the link fails, R1 gets no false hope. R1 correctly knows A is unreachable. The network stabilizes immediately!
